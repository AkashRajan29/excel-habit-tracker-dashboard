📊 Excel Habit Tracker Dashboard
Project Overview

This project is a fully interactive Habit Tracker Dashboard built in Microsoft Excel. The dashboard helps users monitor daily habits, measure consistency, and track progress through automated calculations and visual reports.

The tracker is designed to improve productivity and personal development by providing insights into habit completion rates and performance trends over a 30-day period.

Features

✅ Daily Habit Tracking

✅ Interactive Checkboxes

✅ Automated Completion Percentage

✅ Habit-wise Performance Analysis

✅ Best Habit Identification

✅ Worst Habit Identification

✅ Daily Progress Monitoring

✅ Monthly Performance Dashboard

✅ Dynamic Charts and Visualizations
